
NCU MINI-LISP 直譯器專案 2024
這是一個迷你版的 Lisp，支援基本語法與函式呼叫，並示範遞迴、閉包等概念。
更多詳細語言介紹可參考 PDF 說明或查看原始碼。

功能特色

Basic Features
 -Syntax Validation
 -Print
 -Numerical Operations
 -Logical Operations
 -if Expression
 -Variable Definition
 -Function
 -Named Function

Bonus Features
 -Recursion
 -Type Checking
 -First-class Function


使用方式
安裝 Python 3 與 lark


執行：
python main.py < test.lsp
在 .lsp 檔中撰寫 Mini-LISP 程式碼
更多細節請參考 Mini-Lisp.pdf 與 Compiler Final Project.pdf。
